# AI Industrial Marketer

**B2B Export Marketing Platform** — منصة تسويق تصديري للمصانع التركية.

🌐 **Live Demo:** https://industrial-marketer.agentcraft.info

## What it does

Helps Turkish manufacturers find and reach European B2B clients through:

- **Google Places API** — discovers industrial companies across 81 Turkish provinces
- **Scrapling** — extracts contact emails automatically
- **Claude AI** — writes personalized B2B outreach messages
- **Resend** — sends 100 emails/day automatically

## Tech Stack

- React 18 + TypeScript + Vite
- FastAPI (Python 3.11) Backend
- SQLite Database
- Claude Haiku AI
- Google Places API + Scrapling v0.4.8

## Setup

```bash
cp .env.example .env
# Add your API keys to .env
npm install
npm run dev
```
